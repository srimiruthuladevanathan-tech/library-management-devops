# Online Library Management System with Full DevOps Pipeline
## Objective
Build and deploy a cloud-based Library Management System using a complete DevOps pipeline with GitHub, Docker, Jenkins, Terraform, AWS, and monitoring tools.
## Technology Stack
- Backend: Python Flask
- Frontened: HTML, CSS, JavaScript
- Containerization: Docker
- CI/CD: Jenkins
- Infrastructure: Terraform
- Cloud:AWS
- Monitoring: Amazon CloudWatch
